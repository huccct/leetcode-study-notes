### **题目：**
https://leetcode.cn/problems/contains-duplicate/description/


### **思路：** 
* 首先对数组进行排序
* 比较相邻两个元素是否相等即可得出答案


### **代码:**
```
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function(nums) {
    nums.sort((a, b)=> a - b);
    const n = nums.length;
    for(let i = 0; i < n - 1; i++) {
        if(nums[i] === nums[i + 1]) {
            return true;
        }
    }
    return false;
};
```
