### **题目：**
https://leetcode.cn/problems/binary-search/


### **思路：** 
* 设置默认的idx = -1
* for循环遍历找到对应的target 如果找到可以直接返回 没有就返回idx = - 1


### **代码:**
```js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}
 */
var search = function(nums, target) {
    let idx = -1;
    nums.forEach((item, index) => {
        if(item === target) {
            idx = index;
            return ;
        }
    })
    return idx;
};
```
